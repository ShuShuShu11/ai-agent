# 杭州景点

## 西湖
- **特色**：中国首批国家重点风景名胜区，世界文化遗产，免费开放
- **地址**：杭州市西湖区西湖风景名胜区
- **开放时间**：全天开放
- **建议游览时长**：半天到一天

## 灵隐寺
- **特色**：杭州最著名寺庙，飞来峰造像闻名
- **地址**：杭州市西湖区灵隐路法云弄1号
- **开放时间**：07:00-18:00
- **门票**：75元/人（含灵隐寺+永福寺）

## 千岛湖
- **特色**：因湖中1078个岛屿得名，适合自驾游和亲子游
- **地址**：杭州市淳安县千岛湖镇
- **开放时间**：全天开放
- **门票**：旺季130元/人，淡季110元/人（含船票）
