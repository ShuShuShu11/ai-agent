# 浙江行程规划指南

## 经典5日游路线

### D1：杭州（西湖深度游）
- 上午：断桥残雪 → 白堤 → 曲院风荷
- 午餐：知味观（杭州特色小吃）
- 下午：花港观鱼 → 苏堤 → 雷峰塔
- 晚餐：楼外楼（西湖醋鱼、东坡肉）
- 晚上：观看《印象西湖》演出

### D2：杭州 → 乌镇
- 上午：前往乌镇
- 午餐：乌镇特色菜品
- 下午：东栅景区（百床馆、矛盾故居、木雕馆）
- 晚上：西栅夜景、摇橹船体验
- 住宿：乌镇西栅内民宿

### D3：乌镇 → 苏州（可选） / 千岛湖
- 上午：乌镇西栅深度游
- 午餐后前往千岛湖
- 下午：抵达千岛湖，入住酒店
- 晚上：千岛湖鱼头煲

### D4：千岛湖一日游
- 上午：游船游览中心湖区（梅峰岛、猴岛、锁岛）
- 午餐：船上或湖边渔家乐
- 下午：珍珠广场、秀水街
- 晚上：千岛湖啤酒小镇

### D5：千岛湖 → 返回
- 上午：自由活动（可选择水上项目）
- 午餐后返程

## 周末3日游路线

### D1：杭州
- 下午：抵达杭州
- 傍晚：西湖边散步，欣赏雷峰塔夜景
- 晚餐：外婆家、绿茶餐厅

### D2：杭州西湖一日
- 上午：断桥 → 白堤 → 曲院风荷
- 午餐：知味观
- 下午：花港观鱼 → 苏堤 → 三潭印月
- 晚上：河坊街、南宋御街

### D3：杭州 → 灵隐寺/西溪湿地
- 上午：灵隐寺、永福寺
- 午餐：灵隐素斋
- 下午：西溪湿地（摇橹船体验）
- 返程

## 家庭亲子游路线

### 推荐景点
- 杭州：宋城（演出）、西湖（游船）、动物园
- 绍兴：鲁迅故里、兰亭
- 宁波：罗蒙环球乐园、象山影视城
- 温州：温州乐园、雁荡山

### 推荐美食
- 杭州小笼包、片儿川
- 嘉兴粽子
- 宁波汤团
- 温州鱼丸

## 自驾游路线

### 浙西山水线
杭州 → 临安（青山湖）→ 桐庐（瑶琳仙境）→ 千岛湖

### 浙东海岛线
杭州 → 宁波（天一阁）→ 舟山（普陀山、朱家尖）→ 象山（石浦渔港）

### 浙南生态线
杭州 → 温州（雁荡山）→ 丽水（古堰画乡、云和梯田）

## 出行建议

### 最佳旅游时间
- 春季（3-5月）：西湖苏堤春晓，桃花盛开
- 秋季（9-11月）：桂花飘香，柿子成熟
- 夏季：避暑胜地莫干山、千岛湖

### 交通指南
- 杭州：地铁+公交为主，景区限行
- 城际：高铁网络发达，杭州到各城市很便捷
- 自驾：高速公路发达，适合自驾游

### 住宿建议
- 杭州：西湖边/河坊街周边
- 乌镇：西栅内民宿体验最佳
- 千岛湖：湖景酒店

### 随身物品
- 舒适运动鞋（景区步行较多）
- 防晒用品（夏季）
- 雨具（梅雨季节）
- 常用药品
